Pre-Requisites
1. Technical Knowledge Requirements
To develop and understand the system, the following foundational knowledge is required:
• Programming Languages:
– Python (primary language for AI and backend development)
– JavaScript (for frontend development)
• Artificial Intelligence & Machine Learning:
– Basics of supervised learning
– Understanding of deep learning concepts (CNN, NLP)
– Model training and inference
• Computer Vision:
– Image processing fundamentals
– Face detection and object detection techniques
• Natural Language Processing (NLP):
– Text processing
– Conversational AI basics
• Web Development:
– REST API concepts
– Frontend-backend communication
